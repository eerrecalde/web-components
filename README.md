# Experimenting with web components

This repo contains web components experimental stuff :)

Including (or trying to):
- HTML Imports
- Shadow DOM
- Custom elements
- Templates
- Polyfill

#### Clone the repo:
```
git clone git@github.com:eerrecalde/web-components.git
```

#### Install:
```
npm i
```

#### Run local server:
```
npm start
```

That's it for now.
